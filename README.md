# DAICO contracts

> Smart-contracts of Curio DAICO.

### CurioToken

Base token of Curio DAICO.

- ERC777 interface with ERC20 compatibility
- Minimal granularity, 18 decimals
- 100M total supply

## Usage

### Requirements  

- Node v10.14.1
- NPM v6.9.0

### Install

```
npm i
```

### Compile contracts

```
npm run compile
```

### Run tests

```
npm run test
```
